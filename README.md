# Usage
<!-- usage -->
```sh-session
$ npm install -g comicomi
$ comicomi COMMAND

$ comicomi (-v|--version|version)

$ comicomi --**help** [COMMAND]
USAGE
  $ comicomi COMMAND
...
```
